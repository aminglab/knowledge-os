# Public Cases

This page is the current **public case listing surface** of Knowledge OS.

A public case is not just an article or a static summary.
It is a public release view over governed knowledge objects such as:

- claims,
- evidence,
- dissent,
- verdicts,
- lineage,
- and snapshots.

This listing is still early.
But it already establishes something important:

> public cases are a real category in the public layer, not just one accidental page.

---

## Current public cases

### Power Posing

**Type:** disputed scientific claim  
**Current visible judgment:** the original strong-form claim is contested and significantly weakened; a narrower descendant claim remains contested but still surviving.

**Why this case matters:**
This case makes the central Knowledge OS problem easy to see. A widely circulated scientific claim did not simply remain what it was at publication. It was amplified, challenged, weakened, and partially re-read through a narrower descendant path.

## Why there is only one case here right now

This is the first public case index, not a simulation of a mature public catalog.

The public layer is still in its seeding stage.
So a one-case listing is not a problem as long as it is honest about what it is doing:

- establishing public cases as a category,
- situating the first case as the first member of that category,
- and creating a stable bridge from public entry into case-level reading.
