# Snapshots

This directory will hold frozen publication views derived from the live case objects.

A snapshot is not the full live graph.
It is a release view at a chosen moment.

Future examples may include:

- `snapshot-v0.md` — the first publishable case summary
- `snapshot-v1.md` — a revised public case after additional objects are added

The purpose of keeping this directory now is to make the distinction visible from the beginning:

- **live objects** continue to evolve,
- **snapshots** are deliberate publication releases.
