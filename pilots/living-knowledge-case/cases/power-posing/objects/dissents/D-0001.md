# Dissent D-0001

## Summary

This dissent object represents the later large-sample replication failure led by Ranehill et al. in **2015**.
It directly attacked the core empirical claim associated with the original power-posing result.

## Core details

In the current pilot reading, the key facts are:

- year: **2015**,
- sample size: **200 participants**, roughly five times the original study,
- reported outcome: **no significant effect** on the originally emphasized hormone and behavioral outcomes,
- practical reading: the relevant effect sizes were **near zero**.

## Why this matters

This is one of the clearest moments in the case where the original publication artifact and the later knowledge status diverge.

A static paper can remain stable in circulation.
A living knowledge object must visibly register this challenge.

## Attack surface

This dissent targets the core empirical claim that short power poses produced the originally reported downstream effects on hormone levels and risk-related behavior.

## Consequence for the case

D-0001 is not a side comment.
It is one of the primary reasons the original strong-form claim can no longer be treated as stably supported in this pilot.
