# Evidence E-0001

## Summary

This evidence object represents the initial publication and early support context for the original power-posing claim.

## Core details

In this pilot, the early support context includes at least the following facts:

- the original publication year: **2010**,
- the original reported sample size: **42**, 
- and the reported significance context of the original study: **p < 0.05**.

This pilot also treats the later public amplification of the claim as part of the early support environment, because public visibility shaped how the claim circulated.

## Why this matters

This object is not meant to say that the original publication should be treated as decisive.
It exists so the case can model the first phase of the claim's lifecycle: introduction, publication, visibility, and apparent support.

## Evidence role

E-0001 represents the phase in which the claim entered circulation as if it were robust.
That phase is part of the case, because later dissent does not make sense unless the initial support context is explicitly represented.

## Later refinement

Later versions of this pilot may split this object into:

- the original study itself,
- public amplification,
- and smaller follow-up studies.

For now, they are grouped as one bootstrap support object.
