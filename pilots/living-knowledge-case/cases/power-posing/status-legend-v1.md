# Status Legend v1

This file defines the current **status legend v1** for the power-posing pilot.

Its job is reader-facing rather than object-defining.
It explains how the public wording used in `snapshot-v2.md` should be read, and how that wording is backed by the current verdict grammar layer.

This file therefore sits between:

- the public snapshot wording,
- the case's verdict grammar,
- and the governed claim / verdict objects.

It does **not** create judgment by itself.
It only explains the current public wording surface in a disciplined way.

---

## Why this layer exists

A reader-facing living page should not present status language as if it were decorative prose.

Terms like:

- `contested and significantly weakened`
- `contested but still surviving`

should be understandable as governed status phrases rather than as casual editorial mood.

This layer makes that relationship explicit.
